## GitHub Pages

GitHub Pages 工作流会部署 `14_hideUI` 文件夹，因此线上站点始终使用最终版本。

如果你要部署成：

- `https://<username>.github.io/<repo>/`
  - 仓库名可以自定义。
- `https://<username>.github.io/`
  - 仓库名必须是 `<username>.github.io`。

## Private 说明

仓库是否能保持 private，取决于你的 GitHub 账户套餐与 Pages 支持范围。推送前请在 GitHub 上确认：

- 仓库可见性是否允许 private
- GitHub Pages 是否允许从 private 仓库发布

如果 private Pages 不可用，一个常见做法是：

- 开发仓库保持 private
- 单独建一个 public 的发布仓库或发布分支，只放最终站点内容
